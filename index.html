START法フロー

Simple Triage And Rapid Treatment ― 一次トリアージ

START 1: 歩行可能か？ └ はい → 【緑Ⅲ：軽症（Minor）】 確定 └ いいえ → START 2 へ START 2: 自発呼吸はあるか？ └ なし → 気道確保を行う └ それでも呼吸なし → 【黒 0：死亡・救命困難】 └ 気道確保で呼吸再開 → 【赤Ⅰ：最優先】 └ あり → START 3 へ START 3: 呼吸数は正常か？ └ 30回/分 以上（頻呼吸）→ 【赤Ⅰ】 └ 9回/分 以下（徐呼吸）→ 【赤Ⅰ】 └ 10〜29回/分（正常） → START 4 へ START 4: 循環は保たれているか？ └ 橈骨動脈 触知不可 → 【赤Ⅰ】 └ 橈骨動脈 触れるが微弱 → 【赤Ⅰ】 └ 皮膚の蒼白・冷汗あり → 【赤Ⅰ】 └ 頻脈 120回/分超 → 【赤Ⅰ】 └ 循環不全徴候なし → START 5 へ START 5: 指示に従えるか（従命反応）？ └ 従えない → 【赤Ⅰ】 └ 従える → 【黄Ⅱ：待機的治療】

区分と色

区分色優先度意味
Ⅰ赤最優先生命の危機。直ちに処置が必要
Ⅱ黄待機可処置は必要だが多少待てる
Ⅲ緑軽症歩行可能。処置の緊急性低い
0黒対象外呼吸再開せず。救命困難

PAT法 評価手順

Physiological and Anatomical Triage ― 二次トリアージ（別表第5準拠）

第1段階 生理学的評価（いずれか該当で赤） 意識 : GCS ≤8 または JCS 2桁以上 呼吸 : 10回/分未満 または 30回/分以上 脈拍 : 50回/分未満 または 120回/分以上 血圧 : 収縮期 <90 または ≥200 mmHg SpO2 : 90%未満 体温 : 35℃以下 その他 : ショック症状あり 第2段階 解剖学的評価（いずれか該当で赤） 開放性頭蓋骨骨折・頭蓋底骨折／顔面・気道熱傷 緊張性気胸・開放性気胸・フレイルチェスト 心タンポナーデ／胸腔内出血・胸部臓器損傷 ※所見は疾患名でなく観察所見で表示 ↓第1・第2段階 非該当の場合 基準タグ決定: 歩行可 → 緑 ／ 歩行不可 → 黄 第3段階 受傷機転（該当で1区分上げ） 体幹部・1肢以上の挟圧（4時間以上）／爆発 高所墜落／異常温度環境／NBC災害 第4段階 災害時要援護者（該当で1区分上げ） 小児（≤15歳）／高齢者（≥65歳）／妊婦 基礎疾患患者（心・呼吸器・糖尿病・肝硬変・透析・出血性疾患） ※第3・第4は多重に上げない（1段階のみ） ※緑→黄 ／ 黄→赤 への引き上げのみ（赤→赤は変化なし） ※黒タグは2次対象外
